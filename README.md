# food-truck-app
Food Truck App (Expo + Tailwind)
A simple two-page mobile app built with Expo, Expo Router, and Tailwind CSS. The app includes a Home Screen with the food truck name, tagline, intro textr and featured item, plus a Menu Screen with additional detail. Naviagion works through client side routing, and all styling is done using Tailwind utility classs
